# LockBox

DES algoritmasını kullanarak **paralel veri şifreleme** örneği. 
Bu proje, büyük veri setlerinde şifreleme süresini azaltmak için paralel işlem tekniklerini gösterir. 
Kullanıcılar metin veya dosya üzerinden şifreleme işlemlerini deneyebilir.

## Özellikler
- Metin ve dosya şifreleme
- Paralel işlem optimizasyonu
- Basit kullanım ve güvenli veri işlemleri

## Kurulum
1. Projeyi klonlayın: `git clone <repo_link>`
2. Dist klasörünü açın ve LockBox.exe dosyasını çalıştırın
3. İstediğniz dosyayı seçin ve belirlediğiniz şifre ile şifreleyin
